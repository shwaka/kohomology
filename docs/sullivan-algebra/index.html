Cohomology of Sullivan algebras

In this section, we explain a way to compute cohomology of a Sullivan algebra.

Define a Sullivan algebra​

Consider the minimal Sullivan model of the 2n2n2n-dimensional sphere. Explicitly it is written as (Λ(x,y),d)(\Lambda(x, y), d)(Λ(x,y),d) where the degrees are given by deg⁡x=2n\deg{x}=2ndegx=2n and deg⁡y=4n−1\deg{y}=4n-1degy=4n−1, and the differential is given by dx=0dx=0dx=0 and dy=x2dy=x^2dy=x2.

SphereModel.kt
val n = 2
val indeterminateList = listOf(
    Indeterminate("x", 2 * n),
    Indeterminate("y", 4 * n - 1)
)
val matrixSpace = SparseMatrixSpaceOverBigRational
val sphere = FreeDGAlgebra(matrixSpace, indeterminateList) { (x, _) ->
    listOf(zeroGVector, x.pow(2)) // dx = 0, dy = x^2
}

Compute cohomology​

SphereModel.kt
for (degree in 0 until 10) {
    val basis = sphere.cohomology[degree].getBasis()
    println("H^$degree(S^${2 * n}) = Q$basis")
}

DGA operations​

DGA operations (e.g. sum, multiplication, differential) can be applied within context.run.

SphereModel.kt
val (x, y) = sphere.gAlgebra.generatorList

sphere.context.run {
    // Operations in a DGA can be applied within 'context.run'
    println("d(x * y) = ${d(x * y)}")
    println(d(x).isZero())
    println(x.cohomologyClass())
    println(x.pow(2).cohomologyClass())
}
