MultiDegreeGroup

data class MultiDegreeGroup(val indeterminateList: List<DegreeIndeterminate>) : AugmentedDegreeGroup<MultiDegree>

A AugmentedDegreeGroup representing "parametrized" degrees.

Mathematically, this is equivalent to a direct sum of Z equipped with an augmentation map to Z.
