class CopiedName<D : Degree, I : IndeterminateName>(original: I, shift: D, index: Int?, showShiftExponentInIdentifier: Boolean) : IndeterminateName

An implementation of IndeterminateName representing a shift or duplicate of original.

The option showShiftExponentInIdentifier is provided to the constructor since necessity to print exponent (i.e. uniqueness of exponents) should be decided in the caller of the constructor (e.g. FreeLoopSpace). If it was provided to PrintConfig, it would be impossible to decide.
