In this tutorial, we give some example usages of the library `kohomology`.

## Samples
Samples in this tutorial can be found in [sample](https://github.com/shwaka/kohomology/tree/main/sample/src/main/kotlin/).
To execute `<sampleName>.kt`, run `./gradlew run -DsampleName=<sampleName>` in `sample/`.

## More info
- More examples can be found in [jvmTest](https://github.com/shwaka/kohomology/tree/main/kohomology/src/jvmTest/kotlin/com/github/shwaka/kohomology) as unit tests.
- You can find example projects in [shwaka/kohomology-app](https://github.com/shwaka/kohomology-app).
- Auto-generated documentation can be seen [here](https://shwaka.github.io/kohomology/dokka/index.html) (but currently very few descriptions are given).
